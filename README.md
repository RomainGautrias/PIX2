# Quoi?
L'objet connecté est un système qui vient se visser dans la douille de la lampe.
L'ampoule quant à elle, vient se visser dans l'objet. 

# Pourquoi?
Cet objet permet principalement d'allumer et de régler l'intensité 
lumineuse d'une lampe.

# Pour qui?
Cet objet est destiné au particulier.

# Comment?
Grâce à son smartphone, l'utilisateur contrôle à distance la lampe de la pièce.
Il peut allumer, éteindre ou encore modifier l'intensité lumineuse de la lampe.
